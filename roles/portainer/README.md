Role Name
=========

This role installs the Portainer Community Edition using docker-compose in a given folder.

Requirements
------------

It needs docker-compose and Docker installed in the target host.

Role Variables
--------------

The variables are:

- `portainer_http_port`
- `portainer_https_port`
- `portainer_webapp_port`
